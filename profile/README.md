# BlockChat Organization

🚀 **Добро пожаловать в официальную организацию BlockChat!**

## О проекте

**BlockChat** — это децентрализованный P2P-мессенджер с системой уникальных идентификаторов (BlockChat ID), обеспечивающий максимальную приватность и безопасность общения.

### 🌟 Ключевые особенности:
- **P2P архитектура** — прямое соединение между пользователями без центральных серверов
- **Система BlockChat ID** — уникальные идентификаторы для каждого пользователя
- **End-to-end шифрование** — полная конфиденциальность переписки
- **Кроссплатформенность** — Windows, Linux, macOS
- **Простой интерфейс** — интуитивное управление чатами
- **Поддержка Markdown** — форматирование текста
